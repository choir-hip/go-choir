# Historical Corpus

This directory holds historical designs, missions, reviews, and hypotheses
restored by owner direction on 2026-07-14 from Git snapshot
`2783a97a2221cae812f7eb46de840550a4a802d5`.

It is intentionally visible and searchable. Its value is recall: an older idea
may become useful again, may reveal a discarded alternative, or may supply a
question the current system still has not answered.

It is not current doctrine, an executable Definition, a current architecture
claim, or part of the default reading packet. Before acting on a claim here,
check it against `docs/choir-doctrine.md`, the relevant current domain contract,
the active Definition, and observed code or staging evidence.

This corpus does not restore the pre-purge raw process-evidence or mission-ledger
corpus. Bounded current evidence receipts remain outside this directory; deleted
historical raw detail remains available through Git history when needed.
